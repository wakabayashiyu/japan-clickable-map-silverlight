Japan Clickable Map Silverlight は、都道府県ごとのクリックが可能な日本地図のテンプレートです。
Silverlight (C#)で作られています。

Japan Clickable Map Silverlight is a template of Japanese map in which you can click every prefecture.
It's made using Silverlight (C#).
